Padrões Ethernet

Os padrões Ethernet são uma família de tecnologias de rede de computadores utilizadas em redes locais (LAN).

TCP e UDP

TCP e UDP são protocolos de comunicação utilizados para enviar dados pela internet. TCP é orientado a conexão e confiável, enquanto UDP é sem conexão e mais rápido.
